# Twin-Feeding-Calculator
MIT APP Inventor로 제작

## 목적
쌍둥이 각각의 수유량 체크를 위한 어플. <br>
입력 칸 세개는 맨 위부터 (아기 체중, 마지막 식사 시간, 현재 시간) 이 있습니다. <br>
아기 체중은 Kg 단위로 입력하고, 시간은 HHMM 형식으로 입력하면 됩니다. 현재 시간이 오후 3시 33분이면 1533 을 입력하면 됩니다.
